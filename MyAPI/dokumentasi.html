🧪 Tester 📄 Docs

Dokumentasi REST API

Referensi lengkap endpoint, parameter, dan contoh request & response untuk sistem db_mahasiswa.

Base URL https://jigago.infinityfreeapp.com/MyAPI/

💡 Semua request & response dalam format JSON. Sertakan header Content-Type: application/json pada POST & PUT.
📌 Daftar Endpoint & Fungsi
👤 Mahasiswa — mahasiswa.php
GET/mahasiswa.phpAmbil semua mahasiswa + nama jurusan
GET/mahasiswa.php?id_mhs={id}Ambil satu mahasiswa by ID
POST/mahasiswa.phpTambah mahasiswa baru
PUT/mahasiswa.php?id_mhs={id}Update mahasiswa by ID
DELETE/mahasiswa.php?id_mhs={id}Hapus mahasiswa by ID
🏛️ Jurusan — jurusan.php
GET/jurusan.phpAmbil semua jurusan
GET/jurusan.php?id_jurusan={id}Ambil satu jurusan by ID
POST/jurusan.phpTambah jurusan baru
PUT/jurusan.php?id_jurusan={id}Update jurusan by ID
DELETE/jurusan.php?id_jurusan={id}Hapus jurusan by ID
📚 Mata Kuliah — mata_kuliah.php
GET/mata_kuliah.phpAmbil semua mata kuliah + jurusan
GET/mata_kuliah.php?id_mk={id}Ambil satu mata kuliah by ID
POST/mata_kuliah.phpTambah mata kuliah baru
PUT/mata_kuliah.php?id_mk={id}Update mata kuliah by ID
DELETE/mata_kuliah.php?id_mk={id}Hapus mata kuliah by ID
📊 Nilai — nilai.php
GET/nilai.phpAmbil semua nilai (join mhs & mk)
GET/nilai.php?id_nilai={id}Ambil satu nilai by ID
GET/nilai.php?id_mhs={id}Ambil semua nilai satu mahasiswa
POST/nilai.phpInput nilai baru
PUT/nilai.php?id_nilai={id}Koreksi skor nilai by ID
DELETE/nilai.php?id_nilai={id}Hapus nilai by ID
🔄 Contoh Request & Response
GET Ambil Semua Mahasiswa
Request
GET https://jigago.infinityfreeapp.com/MyAPI/mahasiswa.php
Response — 200 OK
{ "status": { "code": 200, "description": "Menampilkan semua data mahasiswa" }, "result": [ { "id_mhs": 1, "nim": "42510049", "nama_mhs": "Muhammad Nur Yasir Utomo", "alamat": "Jl. Perintis No 1", "umur": 20, "jenis_kelamin": "Laki-laki", "id_jurusan": 1, "nama_jurusan": "Teknik Komputer dan Jaringan" } ] }
POST Tambah Mahasiswa Baru
FieldTipeStatusKeterangan
nimstringrequiredNomor Induk Mahasiswa
nama_mhsstringrequiredNama lengkap
alamatstringrequiredAlamat mahasiswa
umurintegeroptionalUmur
jenis_kelaminstringoptionalLaki-laki / Perempuan
id_jurusanintegeroptionalID Jurusan (FK)
Request
POST https://jigago.infinityfreeapp.com/MyAPI/mahasiswa.php Content-Type: application/json { "nim": "42510099", "nama_mhs": "Budi Santoso", "alamat": "Makassar", "umur": 21, "id_jurusan": 1 }
Response — 200 OK
{ "status": { "code": 200, "description": "1 data mahasiswa berhasil ditambahkan" }, "result": { "id_mhs": 16, /* ... */ } }
PUT Update Mahasiswa
Request
PUT https://jigago.infinityfreeapp.com/MyAPI/mahasiswa.php?id_mhs=1 Content-Type: application/json { "nim": "42510049", "nama_mhs": "Yasir Update", "alamat": "Gowa", "umur": 21, "id_jurusan": 1 }
Response — 200 OK
{ "status": { "code": 200, "description": "Data mahasiswa id=1 berhasil diupdate" } }
DELETE Hapus Mahasiswa
Request
DELETE https://jigago.infinityfreeapp.com/MyAPI/mahasiswa.php?id_mhs=1
Response — 200 OK
{ "status": { "code": 200, "description": "Data mahasiswa id=1 berhasil dihapus" } }
POST Input Nilai Baru
Request
POST http://localhost/Tugas_API/MyAPI/nilai.php Content-Type: application/json { "id_mhs": 1, "id_mk": 2, "skor_nilai": 92 }
Response — 200 OK
{ "status": { "code": 200, "description": "Nilai berhasil ditambahkan" }, "result": { "id_nilai": 47, "skor_nilai": 92 } }
PUT Koreksi Skor Nilai
Request
PUT http://localhost/Tugas_API/MyAPI/nilai.php?id_nilai=1 Content-Type: application/json { "skor_nilai": 95 }
Response — 200 OK
{ "status": { "code": 200, "description": "Nilai id=1 berhasil diupdate" }, "result": { "id_nilai": 1, "skor_nilai": 95 } }
⚡ HTTP Status Codes
200OKRequest berhasil diproses
400Bad RequestField wajib tidak diisi / parameter kurang
404Not FoundData dengan ID tersebut tidak ditemukan
405Method Not AllowedHTTP method tidak didukung endpoint ini
500Server ErrorGagal koneksi database atau query error
